# Agentic Psychology Of Self

This file defines a concise, operational model for agents/subagents in this repository.

## Intent
- Preserve identity and purpose across multi-step coding sessions.
- Improve parent-agent and subagent collaboration.
- Keep creative exploration grounded in measurable engineering outcomes.

## Concept Blend Used
- Psychology of self: active self-monitoring (`I/me`), social feedback loops, and memory continuity.
- True Will: align actions to mission, not impulse; combine freedom with responsibility.
- Tarot/cartomancy: symbolic ideation for visual direction and creative variation only.

- Execution continuity: `plan/progress.md`, `plan/history.md`.

## Working Rule
When a new capability is implemented, update behavior docs and plan artifacts in the same session so memory, identity, and execution remain aligned.

When a new request is broad or identity-shaping, select one deterministic workload first and anchor the session in:
- `plan/blueprints/deterministic-workload-architecture.md`
- `plan/playbooks/deterministic-delivery-playbook.md`
- `plan/runbooks/session-governance-runbook.md`

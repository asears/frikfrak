# Creative Arcana For Pixels

## Intent
Use tarot/cartomancy ideas as creative prompts for visual direction in Frikfrak, especially teletext and retro UI.

## Practical Mapping (Symbol -> Design Move)
- `The Fool`: prototype quickly, low-cost experiments.
- `The Magician`: combine existing assets/tools into new interactions.
- `The Hermit`: diagnostics/debugging screen focus.
- `Wheel of Fortune`: cyclical animation, rotating status states.
- `Justice`: balanced HUD metrics, visible trade-offs.
- `The World`: release-ready integration view.

## Suit Mapping (Minor Arcana -> Engineering Lens)
- `Swords`: logic, static checks, lint/type rules.
- `Cups`: tone, UX copy, player/agent emotional feel.
- `Coins`: performance, memory, reliability budgets.
- `Wands`: momentum, experiments, feature spikes.

## Pixel-Art Prompts
- "Render a 40x25 teletext page themed as `Justice` using error/warn/info/hint columns."
- "Create a cyan-yellow-magenta CRT panel with a `Wheel`-style rotating agent state ring."
- "Design a card-face HUD where each suit corresponds to one build pipeline metric."

## Caution
- Treat tarot as symbolic ideation only.
- Do not claim historical/esoteric facts beyond verified sources.
